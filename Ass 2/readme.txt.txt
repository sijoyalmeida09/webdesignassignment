The following HTML controls were used for this assignment:
1.Favicon:  It displays a favicon image to the left of the page title in the browser tab.
2.Table: It creates table cells inside rows and columns.
3.Images: The HTML <img> tag is used to embed an image in a web page.
4.Hyperlink: Clicking on the hyperlink text, will send the reader to the specified URL address.
5.Button: The <button> tag defines a clickable button.
6.Footer: The <footer> tag defines a footer for a document or section.
7.Float Property: The float property specifies whether an element should float to the left, right, or not at all.
8.Overflow property:The CSS overflow property controls what happens to content that is too big to fit into an area.
9.Relative positioning:An element with position: relative; is positioned relative to its normal position.
10.Tel option:It accepts the recipient’s phone number.
11.Mailto option: It accepts the recipient’s email address.
12.Absolute positioning: an element that is absolutely positioned is taken out of the flow
